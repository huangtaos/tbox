## 功能介绍

tbox 是一个现代化的在线工具箱，提供多种实用工具：

- **开发者工具**：JSON 格式化、Base64 编解码、时间戳转换、正则测试等
- **常用工具**：二维码生成、Markdown 编辑器、PDF 处理、番茄时间、待办事项、饮水追踪等
- **更多工具**：持续更新中...

## 技术栈

- **框架**：Next.js 15
- **前端**：React 19 + TypeScript
- **样式**：Tailwind CSS
- **图标**：Lucide Icons

## 本地运行

### 前置要求

- Node.js 18.0 或更高版本
- npm 或 yarn

### 安装步骤

```bash
# 克隆仓库
git clone https://github.com/huangtaos/tbox.git
cd tbox

# 安装依赖
npm install

# 启动开发服务器
npm run dev
```

访问 http://localhost:3000 查看应用。

### 构建生产版本

```bash
npm run build
npm start
```
